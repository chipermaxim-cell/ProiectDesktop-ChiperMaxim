# Gestionare Biblioteca Personală

Aplicația "Gestionare Biblioteca Personală" este un program desktop care permite utilizatorilor să își organizeze cărțile, să adauge note și să urmărească cărțile citite. Este gândită pentru a fi simplă, intuitivă și ușor de utilizat, astfel încât orice pasionat de lectură să își poată gestiona rapid colecția de cărți.

## Tehnologii planificate
- C#
- .NET 6
- SQLite
- Windows Forms (pentru interfața grafică)
